# WeSplit
Simple bill splitting app to practice SwiftUI views, forms and basic calculations.

## Overview
WeSplit helps split a bill between people and calculates tip amounts. Built while learning layout, forms, and state management in SwiftUI.

## Features
- Enter bill amount, number of people, and tip percentage  
- Live calculation of amount per person  
- Keyboard handling for numeric input

## Concepts & Tech
- SwiftUI Views & Forms  
- `@State`, `@Binding`  
- Simple validation & formatting

## Run
- Xcode: <Version 26.1.1 (17B100)  
- Open `WeSplit.xcodeproj`, build and run.
