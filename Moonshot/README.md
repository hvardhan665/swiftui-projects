# Moonshot
A visually rich SwiftUI app that displays space missions and astronaut details, demonstrating lists, grids, navigation, and JSON decoding.

## Overview
Moonshot showcases advanced SwiftUI layout techniques using mission and astronaut data from a bundled JSON file.

## Features
- Mission grid layout  
- Detailed mission view  
- Astronaut biography screens  
- Adaptive layout based on device size

## Concepts & Tech
- JSON decoding into models  
- NavigationStack  
- LazyVGrid for adaptive layouts  
- Image handling

## Run
- Xcode: <Xcode-version>
- Open `Moonshot.xcodeproj` and run.
