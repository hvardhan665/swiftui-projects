# Guess the Flag
A small game to practice animations, state changes and basic game logic.

## Overview
Players guess country flags from multiple choices; includes score tracking and animations.

## Features
- Multiple-choice flag quiz  
- Score tracking & rounds  
- Animations for correct/incorrect answers

## Concepts & Tech
- `@State` & state-driven UI changes  
- Animations & transitions

## Run
- Xcode: Version 26.1.1 (17B100)
- Open `GuessTheFlag.xcodeproj` and run.
