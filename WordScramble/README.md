# WordScramble
A word game to practice string manipulation, UI updates and game state.

## Overview
Generate scrambled words from letters; score user guesses with rounds and timers.

## Features
- Word generation & scramble algorithm  
- Score & round tracking  
- Time-limited rounds (optional)

## Concepts & Tech
- String & array manipulation  
- Timers & game loop logic  
- State-driven UI updates

## Run
- Xcode: Version 26.1.1 (17B100)
- Open `WordScramble.xcodeproj` and run.
