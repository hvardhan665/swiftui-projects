# LetsConvert
Unit & currency converter demonstrating input handling, segmented pickers, formatting, and optional async networking.

## Overview
LetsConvert allows users to convert between units like temperature, length, and weight â€” with optional currency conversion using an API.

## Features
- Convert between multiple units  
- Clean input handling & formatting  
- Segmented picker for conversion type  
- Optional async currency conversion  

## Concepts & Tech
- TextField input control  
- NumberFormatter  
- Picker & segmented controls  
- Optional async/await networking  

## Run
- Xcode: Version 26.1.1 (17B100)  
- Open `LetsConvert.xcodeproj`, build and run.
