# iExpense
A simple personal expense tracker demonstrating SwiftUI forms, lists, persistence, and model structuring.

## Overview
iExpense allows users to add, categorize, and delete expenses. All data is saved locally using Codable + FileManager for persistence.

## Features
- Add expenses (name, type, amount)  
- Automatically persist expense list  
- Delete expenses  
- Categorized UI with simple styling

## Concepts & Tech
- SwiftUI Lists & ForEach  
- `@StateObject` and ViewModel  
- Codable + FileManager  
- MVVM pattern

## Run
- Xcode: Version 26.1.1 (17B100)
- Open `iExpense.xcodeproj` and run.
