# SwiftUI Projects
A collection of SwiftUI apps built while completing **100 Days of SwiftUI** and through personal learning.

**Repo purpose:** showcase SwiftUI knowledge, MVVM patterns, state management, animations, basic networking, and UI design. Each project has its own small README with screenshots and notes.

## Projects included
- WeSplit  
- BetterRest  
- Guess the Flag  
- Edutainment  
- iExpense  
- LetsConvert  
- Moonshot  
- Cupcake Corner  
- Habit Tracking  
- WordScramble

## Concepts covered across projects
- SwiftUI layout system (Stacks, LazyV/HGrid)  
- State management: `@State`, `@Binding`, `@StateObject`, `@ObservedObject`  
- MVVM structuring for SwiftUI views & view models  
- NavigationStack & sheet presentation  
- Animations & transitions  
- Basic networking with `async/await` (where applicable)  
- Persistence: `UserDefaults`, `FileManager`  
- Accessibility considerations & adaptive layouts

## How to use this repo
1. Open a project folder (e.g., `WeSplit`).  
2. Open the `.xcodeproj` or `.xcworkspace` in Xcode
3. Build & run on simulator or device.
