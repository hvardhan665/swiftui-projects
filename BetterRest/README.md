# BetterRest
Sleep recommendation app that suggests ideal bedtime based on wake-up time, sleep amount and coffee intake.

## Overview
Experiment with SwiftUI forms, DatePicker, and using simple model logic to calculate recommended sleep times.

## Features
- DatePicker for wake-up time  
- Choose desired sleep hours and coffee intake  
- Compute recommended bedtime

## Concepts & Tech
- Date/time handling  
- Form inputs & validation

## Run
- Xcode: Version 26.1.1 (17B100)
- Open `BetterRest.xcodeproj` and run.
