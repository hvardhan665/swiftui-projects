# Habit Tracking
A simple habit tracker demonstrating persistence, model structuring, list management, and optional local notifications.

## Overview
Habit Tracking allows users to create habits, log completions, and track progress over time â€” with persistence across app launches.

## Features
- Add and delete habits  
- Daily completion tracking  
- Persist habits using Codable  
- Optional local reminders  

## Concepts & Tech
- `@StateObject` & ViewModel  
- Codable + FileManager persistence  
- Notification scheduling (optional)  
- SwiftUI list enhancements  

## Run
- Xcode: Version 26.1.1 (17B100)  
- Open `HabitTracking.xcodeproj`, build and run.
