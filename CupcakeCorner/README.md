# Cupcake Corner
A cupcake ordering app demonstrating forms, validation, networking, and data handling using Codable in SwiftUI.

## Overview
Cupcake Corner allows users to select cupcake types, quantities, add delivery details, validate inputs, and optionally send their order to a test API endpoint.

## Features
- Choose cupcake type and quantity  
- Enter delivery information  
- Validate form before submitting  
- Submit order to a test API  
- Display confirmation on success  

## Concepts & Tech
- SwiftUI Forms  
- Codable for JSON encoding/decoding  
- `URLSession` networking  
- `@State` and `@ObservedObject`  
- Simple MVVM structure  

## Run
- Xcode: Version 26.1.1 (17B100)  
- Open `CupcakeCorner.xcodeproj`, build and run.
